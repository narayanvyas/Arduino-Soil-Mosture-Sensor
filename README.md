# Soil Mosture Sensor With Arduino And LCD
Soil Mosture Sensor With Arduino And I2C LCD.

## Prerequisites
1. Arduino Uno R3
2. Arduino IDE
3. I2C LCD
4. Soil Mosture Sensor
5. 220 Ohm Resistor
6. LED
